# isometric-co

Isometric — high-quality carbon removal registry and AI-native certification platform for the industrial economy. Two public OpenAPI 3.1.0 surfaces at `/mrv/v0` (Certify) and `/registry/v0` (Registry), with sandbox and production environments.

- Website: https://isometric.com
- Registry: https://registry.isometric.com
- API docs: https://docs.isometric.com/api-reference/introduction
- Certify OpenAPI: https://api.isometric.com/mrv/v0/mrv.openapi.json
- Registry OpenAPI: https://api.isometric.com/registry/v0/openapi.json

## APIs

- **Isometric Certify Data Ingestion API** — supplier MRV: sources, datapoints, components, removals, GHG statements, monitoring, sensors, feedstocks, biochar applications. 53 paths / 170 schemas.
- **Isometric Registry API** — public registry: issuances, credit batches, balances, transfers, retirements, deliveries, orders, suppliers, projects, Stripe checkout. 38 paths / 79 schemas.
